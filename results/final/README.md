# Final Results

This folder collects the public-facing outputs.

## Main groups

- `final_submission_files/` - manuscript, SI, response letter, and cover letter.
- `water_stress_loss_summary/` - annual and period summary tables.
- `water_stress_loss_summary_detailed/` - detailed tables and the spreadsheet export.
- `fig4_rebuild/` - source tables and exports used to rebuild Figure 4.
- `stress_days_change_maps/` - source data, configs, and maps for future stress-day change.
- `validation_water_stress/` - validation against observed yield-loss anomalies.
- `validation_water_stress_rebuilt/` - rebuilt validation from source data; includes compact validation outputs and a short folder README.

## Note

The very large archival PDF in `validation_water_stress_rebuilt/adm1/1980_2019/` is omitted from GitHub because it is not practical for a normal repository. The compact PDF and PNG versions in the same folder are included instead.
